如何运行机器人 How to start

  1. 配置 cq 目录下的config.yml文件，主目录下的.env .env.dev .env.prod文件。
  2. 用 cmd nb run运行机器人。run bot using nb run .
  3. 用 Windows shell 切换至 cq-http.exe 文件所在目录下， .\go-cqhttp.exe运行。
